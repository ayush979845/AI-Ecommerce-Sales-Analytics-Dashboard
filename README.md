# 🛒 AI Powered E-Commerce Sales Prediction & Analytics Dashboard

## 📌 Project Overview

This project is an **AI-powered E-Commerce Analytics Dashboard** built using **Python and Streamlit**.
It analyzes sales data, visualizes trends, and predicts future sales using **Machine Learning and Deep Learning techniques**

---

## 🚀 Key Features

### 📊 Data Analysis & Visualization

* Dataset preview
* Monthly Sales Trend (Line & Bar Charts)
* Top Countries by Revenue
* Top Selling Products

---

### 🌍 Country-Based Filtering

* Filter data by country
* Monthly sales trend for selected country
* Top products per country

---

### 📦 KPI Metrics

* 💰 Total Revenue
* 📦 Total Orders
* 👥 Total Customers
* 📊 Average Order Value

---

### 🤖 AI Sales Prediction (Machine Learning)

* Model: **Linear Regression**
* Predicts next month's sales
* Graph visualization included

---

### 🧠 Deep Learning Sales Prediction (Local Only)

* Model: **TensorFlow Neural Network**
* Used for advanced prediction
* ⚠️ **Note:** This feature works only in local environment due to cloud limitations

---

### 👥 Customer Segmentation

* Algorithm: **K-Means Clustering**
* Segments customers based on revenue and order count
* Visualized using scatter plot

---

### 🛍️ Product Recommendation System

* Suggests frequently bought products
* Based on transaction history

---

### 💎 Customer Lifetime Value (CLV)

* Calculates customer value:

  * Total Spend
  * Total Orders
  * CLV
* Displays top high-value customers

---

### 📥 Dataset Download

* Download processed dataset directly from dashboard

---

## 🧠 Technologies Used

* Python
* Streamlit
* Pandas
* NumPy
* Matplotlib
* Scikit-learn
* TensorFlow (Local only)

---

## 📂 Dataset

* Online Retail Dataset (CSV format)
* Includes:

  * InvoiceNo, Description, Quantity
  * InvoiceDate, UnitPrice
  * CustomerID, Country

* ✅ Dashboard runs using **app.py**

## ▶️ How to Run the Project

### 1️⃣ Clone Repository

```bash
git clone https://github.com/ayush979845/AI-Ecommerce-Sales-Analytics-Dashboard.git
```

### 2️⃣ Navigate to Folder

```bash
cd AI-Ecommerce-Sales-Analytics-Dashboard
```

### 3️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

### 4️⃣ Run Dashboard

```bash
streamlit run app.py
```
